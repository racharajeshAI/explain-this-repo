Prefer the terminal?

There's a CLI for that. One-time setup, then just paste URLs.

# first time (ask your son to do this, takes 30 seconds)
git clone https://github.com/racharajeshAI/explain-this-repo.git
cd explain-this-repo
pip install -r requirements.txt
# then anytime
python main.py https://github.com/fastapi/fastapi
# output saved as fastapi_explained.md
